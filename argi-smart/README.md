# Argi Smart - Agriculture Education Platform

Modern UI website for agriculture students, farmers, and government services.

## Features
- 🏠 Home page with videos and photos
- 📖 About page
- 🛠️ Services (Rice, Vegetables, Land, Harvesting)
- 📰 News & Updates
- 📞 Contact form
- 🎥 YouTube video integration
- 📱 Responsive design

## Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Run server: `python app.py`
3. Open browser: `http://localhost:8001`
